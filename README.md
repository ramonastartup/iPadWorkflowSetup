# Nova Creative — Landing Page (Bootstrap)

This workspace contains a responsive marketing & creative landing page built with Bootstrap for demo and prototyping.

Files added:

- `index.html` — main landing page
- `css/styles.css` — custom styles layered on Bootstrap

To view locally, open `index.html` in your browser, or serve with a simple static server:

```bash
# from workspace root
python3 -m http.server 8000
# then open http://localhost:8000/index.html
```

The page uses Bootstrap CDN and Bootstrap Icons CDN. Customize copy, images, and the megamenu content as needed.
